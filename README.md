# Projeto de Extensão – Ensino de Python

## Sobre o projeto
Este projeto foi desenvolvido como parte de uma atividade de extensão, com o objetivo de ensinar conceitos básicos de lógica de programação utilizando Python.

## Objetivo
Auxiliar iniciantes a compreender:
- Pensamento computacional
- Lógica de programação
- Estruturas básicas em Python

## Tecnologias
- HTML5
- CSS3

## O que foi feito
- Criação de página simples com HTML e CSS
- Explicação de conceitos básicos de programação
- Apoio no aprendizado de alunos iniciantes
